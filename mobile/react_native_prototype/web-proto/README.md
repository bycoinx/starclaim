StarClaim Web Prototype — Quick Start

Requirements
- Node 18+ recommended

Install & Run
```bash
cd mobile/react_native_prototype/web-proto
npm install
npm run dev
```

What this prototype includes
- Simple React + R3F rotating galaxy background
- HUD placeholders and floating action wheel
- CSS-based Nebula transition overlay for quick UX testing

Notes
- This is a lightweight prototype for UX/motion validation, not the final mobile app.
- For native mobile testing, host the prototype and open on device browser or embed in WebView.
